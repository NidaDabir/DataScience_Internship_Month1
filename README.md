## Week 5: Mini Project – End-to-End Netflix Data Analysis

### Dataset:
Netflix Titles Dataset

### Project Steps:
- Loaded Netflix dataset
- Cleaned missing values (director, cast, country, rating)
- Converted date_added to datetime format
- Extracted year_added column
- Performed Exploratory Data Analysis (EDA)

### Visualizations Created:
- Distribution of Movies vs TV Shows
- Content added per year
- Top 10 countries with most content
- Most common ratings
- Correlation insights

### Business Insights:
- Netflix has significantly more Movies than TV Shows.
- Content addition peaked between 2018–2020.
- United States contributes the highest content.
- TV-MA is the most common rating.
- Rapid content growth happened after 2016.
